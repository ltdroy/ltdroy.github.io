Twinkling Starry Night with JS and Processing (Interactive)

LTD / 16 Feb 2020
Categories: Processing JavaScript
Tags: Casual Algorithmic Art Interactive Visualisation

Background

I’ve been learning about algorithmic art, which is a visual art genre that uses an algorithm (usually coded on a computer) to draw interesting or evocative images. While it is possible to do this in many languages, I decided to delve into the Processing language, which is a Java-style language and development environment that is specialised for visualisation.

What is this?

This is an early casual output of that exploration. It is an interactive visualisation depicting a night sky. The initial stars are placed randomly on the black background, and they twinkle (look carefully!). This essentially works by re-drawing the stars repeatedly, with random-noise added to each draw.

The visual is also interactive, click to add your own stars!
